JavaScript заявления являются командами в браузер

В HTML, заявления JavaScript выполняются в браузере.

JavaScript код последовательность заявлений

Код JavaScript (или просто JavaScript) - последовательность заявлений JavaScript.

JavaScript заявления могут быть сгруппированы в блоки кода
