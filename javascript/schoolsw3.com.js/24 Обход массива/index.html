JavaScript .forEach()

Вызывает функцию один раз для каждого элемента массива.

JavaScript .map()

Создает новый массив, выполняя функцию для каждого элемента массива.

JavaScript .filter()

Создает новый массив со всеми элементами массива, который проходит тест.

JavaScript .reduce()

Проверяет, все ли значения массива прошли проверку.
