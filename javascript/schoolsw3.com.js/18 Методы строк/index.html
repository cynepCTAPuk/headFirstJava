JavaScript определить длину строки

Свойство length возвращает длину строки:

txt.length

JavaScript найти положение текста в строк - indexOf()

Метод indexOf() возвращает позицию первого вхождения указанного текста (поиск производится слева):

str.indexOf("locate")

Метод slice() извлечь часть строки и возвратить извлеченные части в новую строку:

Метод substr() извлечь часть строки и возвращает извлеченные части в новой строке:

Метод substr() извлечь часть строки и возвращает извлеченные части в новой строке:

Заменять "Microsoft" на "Schoolsw3" в параграфе ниже:

str.replace("Microsoft", "SchoolsW3") Пожалуйста посетить Microsoft!
str.replace(/Microsoft/g, "SchoolsW3") Пожалуйста, посетите Microsoft и Microsoft!
str.replace("MICROSOFT", "SchoolsW3") Пожалуйста, посетите Microsoft!
Пример не работает, поскольку MICROSOFT написана заглавными буквами.
str.replace(/MICROSOFT/i, "SchoolsW3") Пожалуйста, посетите Microsoft!
