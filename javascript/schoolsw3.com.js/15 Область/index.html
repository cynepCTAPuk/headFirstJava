JavaScript глобальная область

Доступ к глобальной переменной можно получить из любого скрипта или функции.

JavaScript автоматическая глобальная переменная

Если присвоить значение переменной, которая не была объявлена, она автоматически станет глобальной переменной:

Do not create global variables unless you have planned.
In "Strict mode" automatically global variables will fail.

JavaScript переменная window

В HTML все глобальные переменные станут переменными window.

Do not create global variables unless you intend.

Global variables (or functions) can overwrite window variables (or functions).
Any function, including the window object, can overwrite global variables and functions.
