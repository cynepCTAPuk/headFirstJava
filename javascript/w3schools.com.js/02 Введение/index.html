Изменить содержимое HTML

Что может сделать JavaScript?

JavaScript может изменять содержимое HTML.

Кавычки в коде ' и " можно менять местами друг с другом

JavaScript может изменять значения атрибутов HTML.

В этом случае JavaScript изменяет значение атрибута src (source) изображения.
