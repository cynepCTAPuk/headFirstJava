Задача на программирование: небольшое число Фибоначчи


Дано целое число \( 1 \le n \le 40 \), необходимо вычислить \( n \)-е число Фибоначчи (напомним, что \( F_0=0 \), \( F_1=1 \) и \( F_n=F_{n-1}+F_{n-2} \) при \( n \ge 2 \)).


Sample Input:
3
Sample Output:
2


Memory limit: 256 MB
Time limit: 2 seconds
