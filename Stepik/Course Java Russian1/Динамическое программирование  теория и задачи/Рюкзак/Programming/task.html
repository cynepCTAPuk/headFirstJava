Задача на программирование: рюкзак

﻿

Первая строка входа содержит целые числа \( 1 \le W \le 10^4 \) и \( 1 \le n \le 300 \) — вместимость рюкзака и число золотых слитков. Следующая строка содержит \( n \) целых чисел \( 0 \le w_1, \ldots, w_n \le 10^5 \), задающих веса слитков. Найдите максимальный вес золота, который можно унести в рюкзаке.


Sample Input:
10 3
1 4 8

Sample Output:
9


Memory limit: 256 MB
Time limit: 5 seconds
