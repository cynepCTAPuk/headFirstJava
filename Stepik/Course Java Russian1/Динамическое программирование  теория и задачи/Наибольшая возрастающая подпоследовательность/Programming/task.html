Задача на программирование: наибольшая последовательнократная подпоследовательность

Дано целое число \( 1 \le n \le 10^3 \) и массив \( A[1\ldots n] \) натуральных чисел, не превосходящих \( 2 \cdot 10^9 \). Выведите максимальное \( 1 \le k \le n \), для которого найдётся подпоследовательность \( 1 \le i_1 \lt i_2 \lt \ldots \lt i_k \le n \) длины \( k \), в которой каждый элемент делится на предыдущий (формально: для  всех \( 1 \le j \lt k \), \( A[i_j]\, | \,A[i_{j+1}] \)).


Sample Input:
4
3 6 7 12

Sample Output:
3


Memory limit: 256 MB
Time limit: 5 seconds
