Теоретическая задача повышенной сложности для самостоятельной проверки: локальный минимум за сублинейное время 


Локальным минимумом матрицы называется элемент, значение которого не превосходит значений всех его соседей (считаем, что у внутреннего элемента матрицы четыре соседа, у элемента на границе — три, а у углового элемента — два). Постройте алгоритм, который по данной матрице размера \( n \times n \), содержащей попарно различные натуральные числа, находит (любой) локальный минимум за время \( O(n) \) (такое время называется сублинейным, поскольку оно по порядку меньше размера входа).
