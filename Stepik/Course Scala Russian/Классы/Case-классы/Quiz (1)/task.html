Дан кейс-класс:

case class Dog(name: String, breed: String, ownerName: String)
 И объект этого кейс-класса: 
val bobik = Dog("Bobik", "corgie", "Oleg")
 Олег любит корги и решил завести ещё одну собаку этой породы. Назвать он её хочет Тузик.

 Выберите способы, с помощью которых можно задать представление Тузика в виде кейс-класса.
