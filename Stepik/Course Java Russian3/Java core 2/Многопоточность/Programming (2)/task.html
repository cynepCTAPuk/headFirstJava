Создание гарантированного DeadLock

Необходимо реализовать функции  method1() и method2(): 

  • считается, что job1() и job2() используют общий ресурс;

  • необходимо реализовать двухуровневую блокировку;

  • расположите вызовы блокировок в методах таким образом, чтобы получился deadlock;

  • каждая блокировка в конце метода должна освобождаться, иначе получите минус к карме;

  • нельзя использовать sleep, иначе получите минус к карме;

  • оба метода будут вызваны всего один раз;

  • если Вам надо синхронизировать момент старта в функциях, используйте экземпляр AtomicInteger.


Sample Input:
waitTimeMilliseconds: 1000
Sample Output:
After 1.0 second:
lockA locked
lockB locked
