Безопасный инкремент переменной. Применение блокировок

Необходимо реализовать функцию safeIncrement():

  • в теле функции safeIncrement() производится инкремент переменной count;

  • известно, что функция safeIncrement() одновременно вызывается во множестве потоков;

  • используя экземпляр ReentrantLock, добиться, чтобы конечное значение переменной count было равно количеству вызовов функции safeIncrement().


Sample Input:
thread: 1
Sample Output:
actual: 10000
expected: 10000

Sample Input:
thread: 5
Sample Output:
actual: 50000
expected: 50000


Memory limit: 256 MB
Time limit: 8 seconds
