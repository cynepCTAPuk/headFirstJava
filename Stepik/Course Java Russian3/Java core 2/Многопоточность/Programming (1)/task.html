Безопасный инкремент переменной. Атомики. Практика использования CAS

CAS — compareAndSet, один из самых популярных методов всех атомиков. Попробуйте при помощи CAS сделать макрооперацию атомарной.

Необходимо реализовать функцию safeIncrement():

  • в теле функции safeIncrement() производится инкремент переменной count;

  • известно, что функция safeIncrement() одновременно вызывается во множестве потоков;

  • используя экземпляр AtomicBoolean, добиться, чтобы конечное значение переменной count было равно количеству вызовов функции safeIncrement();

  • блокировками из пакета java.util.concurrent.locks.* и другими средствами синхронизации пользоваться нельзя.

thread: 1
Sample Output:
actual: 10000
expected: 10000

thread: 5
