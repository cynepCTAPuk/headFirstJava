Использование коллектора Collectors.groupingBy 2

Подсчет количества рабочих, занимающих конкретную должность

Есть некоторый Stream с рабочими:

interface Worker {
        String getPosition();
    }

Нужно написать коллектор, который создаст Map, где ключом будет должность, а в качестве значения будет количество рабочих, которые занимают эту должность.



Memory limit: 256 MB
Time limit: 8 seconds
