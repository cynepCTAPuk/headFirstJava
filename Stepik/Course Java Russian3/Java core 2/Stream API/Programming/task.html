Использование filter, map и flatMap, distinct

Необходимо реализовать функцию:

  • на вход функция получает Stream организаций;

  • каждая организация содержит список с рабочими;

  • необходимо вернуть Stream c уникальными именами сотрудников всех организаций, где кол-во сотрудников больше семи;

  • можно пользоваться следующими методами Stream API: filter, map и flatMap, distinct.



Memory limit: 256 MB
Time limit: 8 seconds
