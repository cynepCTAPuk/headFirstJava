Использование коллектора Collectors.joining

Выбор лучших работников в организации

Есть некоторый Stream с рабочими:

interface Worker {
        String getName();
        int getPoints();
    }

Нужно упорядочить сотрудников по количеству очков и выбрать трёх лучших из них.

Нужно написать коллектор, который создаст строку с именами трех лучших сотрудников, разделенными запятой.

Имена сотрудников в строке должны идти в порядке убывания количества очков.

Гарантируется, что количество очков у всех сотрудников разное.



Memory limit: 256 MB
Time limit: 8 seconds
