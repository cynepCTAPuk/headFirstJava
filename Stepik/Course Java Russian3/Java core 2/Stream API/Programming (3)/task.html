Использование коллектора Collectors.groupingBy

Группировка списка рабочих по их должности

Есть некоторый Stream с рабочими:

interface Worker {
        String getPosition();
    }

Нужно написать коллектор, который создаст Map, где ключом будет должность, а в качестве значения будет список рабочих, которые занимают эту должность.



Memory limit: 256 MB
Time limit: 8 seconds
