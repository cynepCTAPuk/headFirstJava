Краткое описание графика работы организации

Необходимо реализовать функцию:

  • на вход функция получает коллекцию с днями недели и временем рабочего дня;

  • формат рабочего дня "09-00, 20-00";

  • если день недели отсутствует в коллекции, то этот день выходной;

  • требуется сформировать строку с кратким графиком работы организации;

  • если два дня подряд идут с одним расписанием, то разделить их запятой, если больше двух дней — дефисом;

  • константы пн–вс и порядок дней взять из класса DayOfWeek;

  • рекомендуемые классы: DayOfWeek, LocalTime, Locale.

Примеры краткого описания организации

пн-сб 09:00-20:00, вс выходной

пн-пт 09:00-18:00, сб, вс выходной

пн-пт 09:00-20:00, сб 10:00-18:00, вс выходной

Sample Input:
5
пн 09:00-20:00
вт 09:00-20:00
ср 09:00-20:00
чт 09:00-20:00
пт 09:00-20:00
Sample Output:
пн-пт 09:00-20:00, сб, вс выходной

Sample Input:
5
пн 09:00-20:00
вт 09:00-20:00
ср 09:00-20:00
пт 09:00-20:00
сб 10:00-19:00
Sample Output:
пн-ср 09:00-20:00, чт выходной, пт 09:00-20:00, сб 10:00-19:00, вс выходной

Sample Input:
6
пн 09:01-20:30
вт 09:01-20:30
ср 09:01-20:30
пт 07:30-15:00
сб 07:30-15:00
вс 07:30-15:00
Sample Output:
пн-ср 09:01-20:30, чт выходной, пт-вс 07:30-15:00
