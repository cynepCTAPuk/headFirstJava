Часовые пояса

Необходимо реализовать функцию:

  • функция принимает локальную дату и время некоторой временной зоны;

  • функция принимает название временной зоны;

  • необходимо эти дату и время преобразовать к московским;

  • рекомендуется применять следующие классы: LocalDateTime, ZoneId, ZoneOffset, Instant.


Sample Input:
2007-05-14T21:27
Europe/Moscow
Sample Output:
2007-05-14T21:27

Sample Input:
2002-01-01T23:55
Europe/Berlin
Sample Output:
2002-01-02T01:55
