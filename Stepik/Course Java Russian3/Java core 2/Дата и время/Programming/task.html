Настройка форматера

Необходимо реализовать функцию:

  • функция должна вернуть экземпляр DateTimeFormatter

  • требуемый формат: 2007-M5-W3-Mon:21:27

  • описание примера требуемого формата: 2007 год, 5 месяц, 3 неделя месяца, понедельник, 21 час, 27 минут

  • посмотреть, как прописывать свой паттерн, можно, например, тут: DateTimeFormatter

  • использовать локаль Locale.UK


Sample Input:
2007-05-14T21:27
Sample Output:
2007-M5-W3-Mon:21:27

Sample Input:
2002-01-01T13:24
Sample Output:
2002-M1-W1-Tue:13:24
