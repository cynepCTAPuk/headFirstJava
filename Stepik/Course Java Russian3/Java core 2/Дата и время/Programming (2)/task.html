Преобразования Instant в Date

Необходимо реализовать функцию:

  • функция принимает время и дату в формате Instant;

  • нужно преобразовать Instant в Date;

  • наносекунды нужно отбросить;

  • если значение Instant больше максимального значения Date, то Date должна принять максимальное значение;

  • если значение Instant меньше минимального значения Date, то Date должна принять минимальное значение.

Такая задача может возникнуть, например, когда Вы работаете с mongoDB.


Sample Input:
2019-10-06T15:13:27.961843900Z
Sample Output:
2019-10-06T15:13:27.961Z
