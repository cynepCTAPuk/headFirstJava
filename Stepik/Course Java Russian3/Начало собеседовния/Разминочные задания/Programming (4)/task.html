Работа с массивом

Необходимо реализовать функцию:

  • на вход функция получает массив целых чисел;

  • нужно определить максимальную длину подмассива, у которого все элементы возрастают,  \(A_{i} \leq A_{i+1}\);

  • вычисленную длину вывести в консоль.


Sample Input:
2 1 4 1 2 3 4 5 6 7 3 1
Sample Output:
7


Memory limit: 256 MB
Time limit: 8 seconds
