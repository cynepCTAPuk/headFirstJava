Postman

Это программа, которая предоставляет удобный графический интерфейс для создания различных http-запросов.

При разработке сервера с REST API для ручных тестов рекомендую использовать подобный софт.

Установите Postman на свой компьютер и попробуйте сделать какие-нибудь запросы.

https://any-api.com/ – содержит перечень открытых api, которые можно подергать руками. Выберите сервис, прочитайте на него документацию и попробуйте получить при помощи Postman какие-нибудь результаты.
