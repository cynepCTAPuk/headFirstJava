Протокол HTTP

Методы HTTP

Основные методы, которые применяются везде, – это post и get.

get – используется для получения содержимого указанного сетевого ресурса. Как правило, get-запросы не изменяют состояние сервера. Get-запрос никогда не содержит тело.

post – используется для передачи данных указанному сетевому ресурсу или для запуска управляющих воздействий.

Методы HTTP
get options
post head
put trace
patch connect
delete  

Группа методов get, post, put, patch, delete используется для чтения, сохранения, обновления и удаления указанного сетевого ресурса.

Группа методов options, head используется, чтобы получить информацию о сервере.

Группа методов trace, connect используется для прокси и получения дополнительной информации о сети.

Статусы ответов HTTP

Группы кодов Описание
1xx группа информационных кодов
2xx коды, указывающие на успех запроса
3xx указывают, что нужно сделать перенаправление
4xx ошибка клиента
5xx ошибка сервера
