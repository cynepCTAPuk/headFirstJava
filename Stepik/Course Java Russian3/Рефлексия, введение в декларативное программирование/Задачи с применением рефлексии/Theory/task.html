Для этого задания Вам потребуется подключить в Ваш проект библиотеку для генерации отчетов в Excel — Apache POI.

Вам нужно разработать библиотеку для генерации отчетов в Excel поверх библиотеки Apache POI.

Ваша библиотека должна уметь открывать Excel-шаблоны вида:

где ~{nameColumn} — ячейка с шаблоном (координаты колонки, стили ячейки) для колонки с названием nameColumn.

Вашей библиотеке передаются списки объектов вида:

class Worker {
        @ExelColumn("name")
        private String name;

        @ExelColumn("salary")
        private int salary;

        @ExelColumn("birthday")
        private LocalDate birthday;

    }
