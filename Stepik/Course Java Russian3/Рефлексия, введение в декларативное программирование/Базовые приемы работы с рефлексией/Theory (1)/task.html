Примечание к задачам

Вы не сможете добавить свои импорты в тестируемый код Java.

Вместо этого во всех задачах уже добавлены следующие импорты:

import java.io.*;
import java.lang.reflect.*;
import java.lang.annotation.*;
import java.util.*;
import java.util.stream.*;
import java.util.function.*;
import java.math.*;
