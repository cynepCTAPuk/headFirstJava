Получение названий всех полей класса

Необходимо реализовать функцию:

  • на вход функция получает class definition;

  • необходимо определить и вывести в консоль все наименования полей класса;

  • наименования полей класса должны быть отсортированы по алфавиту.

Описание тестового класса A:

class A {
        final int age;
        final String name;

        public A(int age, String name) {
            this.age = age;
            this.name = name;
        }
    }

Sample Input:
class A
Sample Output:
age
name


Memory limit: 256 MB
Time limit: 8 seconds
