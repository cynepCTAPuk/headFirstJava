Вызов метода средствами рефлексии

Необходимо реализовать функцию: 

  • на вход функция получает объект;

  • на вход функция получает название метода;

  • на вход функция получает аргументы вызываемого метода;

  • требуется у объекта вызвать метод с соответствующим названием;

  • гарантируется, что у объекта названия всех методов уникальны.

Пример объекта с методом, который должен будет вызываться:

class A {
        public void sayHello() {
            System.out.println("Hello world!!!");
        }
    }

Подсказка:

Проще всего взять у класса все методы и из них выбрать подходящий по названию.


Sample Input:
methodName sayHello
Sample Output:
Hello world!!!

Sample Input:
methodName sayHello2 the best world
Sample Output:
Hello the best world!!!


Memory limit: 256 MB
Time limit: 8 seconds
