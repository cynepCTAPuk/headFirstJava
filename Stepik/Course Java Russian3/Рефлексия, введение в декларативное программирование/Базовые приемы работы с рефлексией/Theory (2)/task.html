MethodHandler

Самостоятельно реализуйте задачу, описанную в шаге "Вызов метода средствами рефлексии", только при вызове метода вместо method.invoke(object) воспользуйтесь MethodHandler.

Дополнительно

  • прочитайте результат выполнения метода
  • для метода, возвращающего String, результат запишите в Object (без промежуточных переменных)
  • чем MethodHandler лучше по сравнению с method.invoke(object, params)?
