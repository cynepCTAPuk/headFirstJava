Создание proxy

Необходимо реализовать функцию: 

  • на вход функция получает объект;

  • необходимо создать proxy для данного объекта;

  • гарантируется, что у всех вызываемых методов отсутствуют аргументы;

  • в прокси-классе должны дублироваться вызовы методов;

  • для генерации прокси класса использовать - Proxy.newProxyInstance;

  • если метод аннотирован аннотацией @Logging, то перед вызовом метода вывести в консоль "Before call MethodName";

  • если метод аннотирован аннотацией @Logging, то после вызова метода вывести в консоль "After call MethodName".

Аннотация @Logging:

@Target(ElementType.METHOD)
    @Retention(RetentionPolicy.RUNTIME)
    public @interface Logging {
    }

Пример объекта:

class A implements Base {
        @Logging
        public void method1() {
            System.out.println("A method1");
        }

        public void method2() {
            System.out.println("A method2");
        }
    }

    interface Base {
        void method1();
        void method2();
    }

Sample Input:
class A
Sample Output:
Before call method1
A method1
After call method1
A method2
