Получение аннотаций и их параметров

Необходимо реализовать функцию: 

  • на вход функция получает объект;

  • необходимо вызвать все методы, помеченные аннотацией @Weight, в порядке возрастания их весов;

  • гарантируется, что значения весов у методов уникальны;

  • гарантируется, что у всех вызываемых методов отсутствуют аргументы.

Аннотация @Weight:

@Target(ElementType.METHOD)
    @Retention(RetentionPolicy.RUNTIME)
    public @interface Weight {
        int value();
    }

Пример объекта:

class A {
        @Weight(10)
        void method1() {
            System.out.println("method1");
        }

        @Weight(3)
        void method2() {
            System.out.println("method2");
        }

        @Weight(21)
        void method3() {
            System.out.println("method3");
        }
    }

Sample Input:
class A
Sample Output:
method2
method1
method3
