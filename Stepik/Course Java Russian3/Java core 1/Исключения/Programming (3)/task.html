ConcurrentModificationException

Реализуйте функцию:

  • принимает ArrayList<Integer>, который в это время параллельно модифицируется в другом потоке;
  • напишите произвольный код, который будет работать с данным ArrayList<Integer>;
  • именно Ваш участок кода должен сгенерировать ConcurrentModificationException;
  • нельзя использовать оператор throw.
