Как устроен класс String изнутри

 

java 7,8
Название поля Описание
int hash cache переменная с хешем строки
char value[] массив символов строки

hash, пока у строки ни разу не вызван метод hashCode, равен нулю.

Таким образом, реальный хеш для строки высчитывается 1 раз.

 

java 9+
Название поля Описание
int hash Кеш переменная с хешем строки
byte[] value Массив символов строки
byte coder Указывает на кодировку текста (UTF-16 или Latin-1)

В зависимости от кодировки символы будут занимать один или два байта.

Компактные строки 1 байт/символ предназначены исключительно для англоязычного текста.
