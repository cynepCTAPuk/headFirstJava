Собственный класс для больших чисел

Необходимо реализовать класс:

  • в конструктор передается строка с числом в десятичной системе счисления;

  • числа могут быть как положительными, так и отрицательными;

  • реализовать методы для сложения и вычитания таких чисел;

  • числа должны быть иммутабельны;

  • реализовать метод compareTo;

  • нельзя использовать класс BigInteger и подобные.
