Реверс байтов

Необходимо реализовать функцию:

  • на вход функция получает число;

  • внутри функции нужно реализовать реверс байтов;

  • функция должна вернуть новое число;

  • запрещено пользоваться функцией Integer.reverseBytes;

  • рекомендуется пользоваться битовыми операциями.


Sample Input:
0xfe01ccd1
Sample Output:
0xd1cc01fe


Memory limit: 256 MB
Time limit: 8 seconds
