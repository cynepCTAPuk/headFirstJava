Комплексные числа

Реализовать иммутабельный класс комплексных чисел, в котором:

  • есть операции сложения, вычитания и умножения;
  • реализованы корректные методы equals и hashCode;
  • реализован toString, формат - A+Bi, где A - действительная часть, B - мнимая часть, точность - до двух знаков после запятой;
  • создание нового комплексного числа подразумевается только фабричным методом createComplex;
  • два комплексных числа считать равными, если их действительная и мнимая части соответственно равны с точностью до двух знаков;
  • для методов equals и hashCode рекомендуется использовать строковое представление.

Sample Input:
1 1
7 -3.321
Sample Output:
sum: ok
sub: ok
mul: ok
equals and hashCode: ok

Sample Input:
6.13 71812.1221
-9.12121212 -0.121121
Sample Output:
sum: ok
sub: ok
mul: ok
equals and hashCode: ok
