Класс Object

Название метода Описание
equals проверяет, равен ли один объект другому
hashCode числовая функция, используется для быстрого сравнения объектов
toString возвращает строку, описывающую объект
getClass возвращает спец. класс с описанием данного класса
finalize вызывается перед удалением объекта сборщиком мусора
clone используется для создания копии объекта
wait ожидает, пока другой поток не разбудит данный поток
notify будит другой поток, спящий на wait
notifyAll будит все потоки, спящие на wait

Типичный вопрос на собеседовании: назовите все методы класса Object.

Методы equals и hashCode неразрывно связаны. Существует правило: два равных объекта должны иметь одинаковый hashCode, но при этом два объекта, у которых равный hashCode, не обязательно равны между собой.

Самые часто используемые методы: equals, hashCode, toString, getClass.

Считается, что все классы в Java унаследованы от класса Object, даже если у класса явно не указан родитель.
