Коллекции

Под коллекциями в Java понимают контейнеры для групп однотипных элементов.

Что нужно знать:

  • иерархию основных коллекций;
  • что такое итератор;
  • как правильно выбирать коллекцию под ситуацию;
  • строение ArrayList, LinkedList, плюсы и минусы этих коллекций;
  • строение HashMap, зачем нужна иммутабельность ключа;
  • время выполнения типовых операций в О-нотации для таких операций, как add, put, get.

Для описания производительности операций у коллекций, как правило, используют О-нотацию:

  • О(1) - константное время (добавление эл-та в конец ArrayList);
  • О(n) - время, пропорциональное количеству элементов (добавление эл-та в начало ArrayList);
  • О(log(n)) - время, логарифмически пропорциональное количеству элементов (добавление эл-та в TreeSet).
