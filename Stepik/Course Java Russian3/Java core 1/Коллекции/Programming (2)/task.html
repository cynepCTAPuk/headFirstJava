Определение родителя для узла в дереве

Необходимо реализовать функцию:

  • функция принимает List с элементами дерева, причем гарантируется, что все они принадлежат одному дереву;
  • у элементов не заполнено поле parent, необходимо это поле заполнить для каждого элемента, кроме корня.
class Node {
        Node left;
        Node right;

        Node parent; // у всех элементов равно null, надо заполнить
    }
