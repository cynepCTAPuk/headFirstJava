Создание экземпляров подставляемого типа T

Необходимо реализовать класс-фабрику:

  • класс обобщен параметром T;

  • необходимо реализовать функцию, которая будет создавать экземпляры класса T;

  • в задании применить методы рефлексии;

  • гарантируется, что у класса Т есть конструктор по умолчанию;

  • если Вам будет сложно, вернитесь к заданию после прохождения модуля по рефлексии.

Пример целевой фабрики для класса:

class FactoryA extends Factory<A> {

    }

Рекомендуемая операция для создания экземпляра:

Class<T> aClass = ...

aClass.getDeclaredConstructor().newInstance()

 

Импортированные библиотеки в этом задании:

import java.io.*;
import java.lang.reflect.*;
import java.lang.annotation.*;
import java.util.*;
import java.util.stream.*;
import java.util.function.*;
import java.math.*;

Sample Input:
className A
Sample Output:
Expected instance: {A}, Actual instance: {A}

Sample Input:
className B
Sample Output:
Expected instance: {B}, Actual instance: {B}
