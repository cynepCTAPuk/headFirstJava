Generics

Под Generics в Java понимают способ задания обобщенных классов и методов.

class Box<T> { // Generic type
        private final T value;

        Box(T value) {
            this.value = value;
        }

        static <R> Box<R> of(R value) { // Generic method
            return new Box<>(value);
        }
    }
    
    public static void main(String[] args) {
        Box<String> stringBox = new Box<>("str");
        Box<Integer> integerBox = new Box<>(42);

        Box<Integer> integerBox2 = Box.of(42);
        Box<String> stringBox2 = Box.of("str");
    }

В угловых скобках <> задается метасимвол, который обозначает подставляемый тип.

При задании generic типа или метода можно ограничить подставляемые типы базовым родителем при помощи слова extend.

class Box<T extend Number>{
  // class definition
}

Специальный метасимвол ?, также называемый wildcard, обозначает любой тип. 

Box<?> box = new Box<>(42);

При создании экземпляра generic класса для создания диапазона подставляемых типов можно использовать вместе с символом ? слова extend и super.

Box<? super Number> box = new Box<>(42); // Integer
box = new Box<>(42.0); // Double
