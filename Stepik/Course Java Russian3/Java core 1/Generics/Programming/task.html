Стирание типов

Необходимо реализовать функцию:

  • на вход функция получает вещественное число;

  • функция возвращает результат с типом Box<Integer>;

  • добейтесь, чтобы функция возвращала Box<Integer>, где value будет на самом деле типа Double;

  • для нового экземпляра Box необходимо использовать входной параметр.
